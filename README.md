# 📄 Research Paper QA System (RAG)

## 🚀 Overview
A Retrieval-Augmented Generation (RAG) based Question Answering system built on research papers.

## 🧠 Features
- Semantic search using embeddings
- FAISS vector database
- LLM-based answer generation
- Source citation

## 🏗️ Tech Stack
- Python
- Sentence Transformers
- FAISS
- Streamlit
- LangChain (Phase 2)
